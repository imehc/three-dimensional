## 核心概念

- **Model类架构**: 将场景管理、相机、渲染器等分离为独立的系统
- **GLTF模型**: 加载包含动画数据的3D模型文件
- **AnimationMixer**: 管理和播放模型内的动画剪辑
- **OrbitControls**: 实现围绕目标点的相机交互
- **系统设计**: 使用Loop、Resizer等系统管理渲染和响应式布局
