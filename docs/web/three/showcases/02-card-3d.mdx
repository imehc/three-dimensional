## 核心概念

- **Physics世界**: 使用Rapier物理引擎模拟真实物理效果
- **RigidBody**: 参与物理模拟的刚体，有动态和静态两种
- **约束系统**: RopeJoint和SphericalJoint用于连接刚体，创建绳索效果
- **MeshLine**: 使用贝塞尔曲线渲染光滑的线条
- **交互系统**: 指针事件和鼠标拖拽实现3D交互
