## 核心概念

- **Html**: @react-three/drei 提供的组件，用于在 3D 场景中嵌入 HTML 元素
- **scene.traverse()**: 递归遍历场景中的所有对象
- **userData**: Three.js 对象上存储自定义数据的属性
- **distanceFactor**: 控制 HTML 元素随相机距离的缩放行为
- **缓存机制**: 避免重复生成 JSX，提高性能
