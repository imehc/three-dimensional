## 核心概念

- **useGLTF**: @react-three/drei 提供的 Hook，用于加载和缓存 GLTF/GLB 模型
- **自动缓存**: 已加载的模型会被缓存，再次请求相同 URL 时返回缓存版本
- **动态切换**: 可以根据用户输入动态更改模型 URL
- **Leva 选择器**: 使用 Leva 的 options 功能创建下拉菜单
