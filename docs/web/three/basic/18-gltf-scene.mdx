## 核心概念

- **ContactShadows**: 生成接触阴影，增强物体与地面的互动感
- **Leva**: UI 控制库，用于实时调整参数并在浏览器中显示控制面板
- **useControls**: Leva 的 Hook，定义可控制的参数
- **PBR 材质**: 基于物理的渲染（Physically Based Rendering），提供逼真的材质效果
