# Theatre 编辑器

<App/>

## 核心概念

- **Theatre.js**: 强大的动画库，提供关键帧编辑和播放功能
- **@theatre/r3f**: Three.js 集成库，使 Three.js 对象支持 Theatre.js 编辑
- **editable 组件**: 支持在 Theatre.js 编辑器中实时编辑属性（位置、旋转、缩放等）
- **SheetProvider**: 提供动画上下文，使子组件可以访问动画表单数据
