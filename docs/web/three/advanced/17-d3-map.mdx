## 核心概念

- **D3.js geoMercator**: 地理投影函数，将经纬度坐标转换为平面坐标
- **ExtrudeGeometry**: 将 2D 形状拉伸成 3D 立体几何体
- **GeoJSON**: 地理信息编码格式，包含多边形和点位等地理数据
- **MultiPolygon vs Polygon**: 地理数据的两种类型，需要分别处理
